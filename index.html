Projet Info 2A

Agrégateur de flux RSS et Tweets

Rappel du besoin

Réaliser un agrégateur de flux RSS et de tweets pour suivre l'actualité

Fonctionnalités de base

  • Inscription
  • Récupérer des flux RSS
  • Récupérer des tweets
  • Organiser cela en thématique
  • Permettre aux utilisateurs de s'abonner à des thématiques
  • Pouvoir modifier des thématiques directement via l'application
  • Sauvergarder, annoter une actu

Fonctionnalités de avancées

  • Permettre à un utilisateur de partager sur son twitter une actu/un tweet
  • Envoyer une notif aux utilisateurs quand une thématique est modifiée
  • Ajouter des fonctionnalités de réseau social (commentaire d'actu, like, laisser les utilisateurs poster une actu, etc)

Application Programming Interface

Que nous dit wikipedia

En informatique, une interface de programmation applicative (souvent désignée par le terme API pour application programming interface) est un ensemble normalisé de classes, de méthodes ou de fonctions qui sert de façade par laquelle un logiciel offre des services à d'autres logiciels.

Dit autrement

  • Brique applicative qui offre un service, et qu'on utilise sans savoir comment elle fonctionne
  • 2 types
    • Les bibliothèques logicielles
    • Les web services

  • Note : vous risquez d'utiliser les deux

L'api twitter 1/2

  • Authentification d'un compte tier
  • Récupérer des tweets
  • En poster
  • Direct message
  • ...

Et tout ça simplement en appelant une simple URL

L'api twitter 2/2

  • Renvoie des json (JavaScript Object Notation)
