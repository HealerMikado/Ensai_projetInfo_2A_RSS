Episode spin-off : Les proxys (HTTP)

Les proxys (HTTP)

"Juste" un intermédiaire entre un pc/réseau et internet.

"Être caché derrière un proxy"

Configurer un proxy sur son navigateur

  • Normalement c'est déjà fait
  • Pour les curieux Firefox>option>avancé>réseau

Mais pour le reste, faut le faire vous même (pip install, python ...)

Les avantages

  • Sécurisation
  • Filtrer les sites
  • Logger les échanges
  • Optimiser la navigation

Les inconvenients

  • Faut le configurer
  • Une seule IP pour tout l'Ensai
  • Pirater le proy compromet tout le réseau
